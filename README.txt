Tool to calculate macro-nutrients in a meal. Asks for ingredients and the ammount used in grams, then exports macro-nutrient data to excel.

To get started, install pip (Helpful link: https://www.liquidweb.com/kb/install-pip-windows/) and then run 'pip install -r requirements.txt' in your terminal. Afterwords, run 'pip install nutritionix. To execute the program, type 'python3 macroCalc.py'
